# منصة الناصر التعليمية | نظام الباحث القانوني المتطور

منصة رقمية متكاملة متخصصة في تأهيل وتدريب الباحثين القانونيين، تم تطويرها وتصميمها بواسطة **أ. معين الناصر**. تعتمد المنصة على أحدث التقنيات البرمجية لتوفير تجربة اختبارات تفاعلية وسلسة.

## 🚀 نظرة عامة على المشروع
تم بناء هذه المنصة لخدمة القطاع التعليمي القانوني، حيث تتيح للطلاب والممارسين الوصول إلى بنوك أسئلة محدثة ونماذج اختبارات دورية تحاكي الواقع.

## 🛠 التقنيات المستخدمة (Tech Stack)
تم تطوير المشروع باستخدام بنية برمجية حديثة تضمن السرعة والأمان:
- **Framework:** React.js (Vite)
- **Language:** TypeScript
- **Styling:** Tailwind CSS & shadcn/ui
- **Backend & Database:** Supabase
- **Deployment:** Vercel

## ✨ المميزات 
- **نظام اختبارات ذكي:** محرك اختبارات يختار الأسئلة عشوائياً بناءً على السنة والمادة.
- **لوحة تحكم إدارية:** نظام كامل لإدارة الأسئلة، المواد، والمستويات الدراسية.
- **تحليلات الأداء:** تقارير فورية للطلاب مع مراجعة شاملة للإجابات الصحيحة والخاطئة.
- **توافق كامل:** تصميم متجاوب يعمل بكفاءة عالية على كافة أجهزة الجوال والحواسيب.

## 💻 التشغيل المحلي
للمطورين الراغبين في تشغيل نسخة محلية من المشروع:
1. `git clone` للمستودع.
2. `npm install` لتثبيت المكتبات.
3. `npm run dev` لتشغيل خادم التطوير.
